Вариант 4: работа со временем (часы, минуты, секунды).
Программа позволяет вычислить количество полных минут во введённом времени,
а также уменьшить время на 10 минут.

Приложение реализовано на WPF (C#) с использованием контейнера Grid,
обработчиков событий и валидации вводимых данных.
